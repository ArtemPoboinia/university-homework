# Задание: Составить число

## Описание
Программа находит последовательность операций сложения и умножения, позволяющую получить заданное число `num` из `1`.

## Входные данные
- `num` (целое число) - целевое число
- `a` (целое число) - число для операции сложения
- `b` (целое число) - число для операции умножения

## Выходные данные
- Последовательность операций `+a` и `*b`, преобразующая `1` в `num`
- Если решение невозможно, выводится `Путь не найден`

## Алгоритм
Используется рекурсивный обратный поиск:
- Начинаем с целевого числа `num`
- На каждом шаге проверяем:
  1. Если `num % b == 0`, пробуем делить на `b` (обратная операция умножению)
  2. Если `num - a >= 1`, пробуем вычитать `a` (обратная операция сложению)
- Рекурсивно продолжаем, пока не достигнем `1` или не докажем невозможность решения
