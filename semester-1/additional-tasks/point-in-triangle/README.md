# Задание: Точка в треугольнике

## Описание
Программа проверяет, лежит ли точка в треугольнике.

## Входные данные
- `x1` (double) - координата x первой точки
- `y1` (double) - координата y первой точки
- `x2` (double) - координата x второй точки
- `y2` (double) - координата y второй точки
- `x3` (double) - координата x третьей точки
- `y3` (double) - координата y третьей точки
- `px` (double) - координата x точка
- `py` (double) - координата y точки

## Выходные данные
- При вводе не правильных координат программа выводит: `Ошибка! Введите число: ` и пробует заново ввести их
- Если треугольник с введёнными координатам не существует, то программа выводит: `Треугольник не существует`
- Если точка лежит в треугольнике то программа выводит: `Точка лежит ВНУТРИ треугольника`
- Если точка не лежит в треугольнике то программа выводит: `Точка НЕ лежит в треугольнике`

## Алгоритм
- Ввод данных
- Каждая координата проходит проверку на верность
- Происходит проверка на существования треугольника
- Вычисляется площадь всех треугольников
- Происходит проверка: лежит ли точка в большом треугольнике
Суть проверки:
- Если площадь большого треугольника - сумма площадей маленьких треугольников > epsilon
- Точка лежит в треугольнке
